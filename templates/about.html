About SpyVision

SpyVision is an AI powered hidden camera detection system developed using Convolutional Neural Networks (CNN). The system scans live webcam input and identifies whether the detected device is a spy camera or a normal camera.

The model is trained using image datasets and enhanced using GAN based data augmentation techniques to improve detection accuracy.

Technologies Used

  • TensorFlow
  • OpenCV
  • CNN Deep Learning Model
  • GAN Dataset Augmentation
